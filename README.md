# Leo Recovery - Лендинг приложения

Современный одностраничный лендинг для iOS приложения Leo Recovery.

## Быстрый старт

1. Откройте `index.html` в браузере
2. Готово!

## Структура

```
Leo Recovery(сайт)/
├── index.html
├── styles.css
├── script.js
└── assets/rusMocups/
```

## Публикация

### GitHub Pages
```bash
git init
git add .
git commit -m "Initial commit"
git push origin main
```
Settings → Pages → Deploy from main

### Netlify
Перетащите папку на netlify.com/drop

## Редактирование

**Тексты:** index.html  
**Цвета:** styles.css → :root  
**Изображения:** assets/rusMocups/

## Технологии

- HTML5
- CSS3 (Glassmorphism, Grid, Flexbox)
- JavaScript (Intersection Observer, Animations)
- Google Fonts (Inter)
